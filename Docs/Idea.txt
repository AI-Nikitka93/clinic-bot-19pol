ONE-LINER

Интеллектуальная система мониторинга электронной записи в поликлинику, которая в реальном времени отслеживает появление, исчезновение и изменение талонов и мгновенно уведомляет пользователей по заданным критериям.

ORIGINAL IDEA / RAW INPUT

Создать Telegram-бота для сайта записи в поликлинику, который постоянно отслеживает наличие талонов. Бот должен реагировать не только на появление новых талонов, но и на освобождение уже занятых, отмены записей, открытие новых дней и любые изменения расписания. Пользователь может подписаться как на всех врачей определенной специальности, так и выбрать конкретного врача, например своего участкового терапевта.

IDEA SUMMARY

Проект представляет собой сервис непрерывного мониторинга системы электронной записи в поликлинику с гибкими пользовательскими фильтрами, историей изменений, аналитикой и мгновенными уведомлениями.

Главная задача — избавить пользователя от необходимости вручную обновлять сайт в поисках свободного талона.

FINAL GOAL

Создать надежную систему, которая самостоятельно отслеживает все изменения в расписании врачей, сравнивает их с предыдущими состояниями и уведомляет пользователя только о действительно важных событиях.

CORE PROBLEM

Пользователь вынужден постоянно самостоятельно проверять сайт.
